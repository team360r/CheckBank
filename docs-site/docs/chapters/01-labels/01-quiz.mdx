  <QuizQuestion
    question="When should you use MergeSemantics?"
    options={[
      "When you want to hide a widget from screen readers entirely",

  <QuizQuestion
    question="What does ExcludeSemantics do, and when should you use it?"
    options={[
      "It hides a widget visually but keeps it in the semantics tree",
      "It removes a widget and all its descendants from the semantics tree — use it for decorative elements that add noise for screen reader users",
      "It marks a widget as important for screen readers to announce first",
      "It disables tap actions on a widget for accessibility users"
    ]}
    correctIndex={1}
    explanation="ExcludeSemantics removes the wrapped widget from the semantics tree entirely. Screen readers will never encounter it. Use it for decorative icons, background images, and any visual element that repeats information already available in adjacent text. Without it, decorative elements create noise that slows down navigation."
  />
